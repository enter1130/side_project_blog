# Blog Side Project
該project是利用laravel與react開發，主要用於實作是rsuitejs、編寫文章、留言、CRUD等。

# 2023-6-28 v1.0.0

- 新增留言功能
- 新增回覆留言功能
- 新增刪除留言功能
- 新增喜歡留言功能
- 新增查閱人數顯示、按讚顯示等功能
- 新增登入功能
